## Tech Stack

- **UI:** shadcn/ui, Tailwind CSS

- **Database:** PostgreSQL (Supabase/Neon)
- **ORM:** Prisma
- **Auth:** Clerk
- **Queue:** Bull + Redis (Upstash)
- **Payments:** Paddle
- **Hosting:** Vercel
